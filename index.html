Twitter Time Series Data Visualization

Using GeoTools Java Library

The objectives of the assignment was to establish WMS connection, request WMS image of user defined bounding box, store image locally into KML structure, display the stored image in google earth , visualize the tweets as a time series overlayed on the image. The code consist of three classes:

  • WMS Connection: Establishes the connection with the WMS, assigns the intended parameters, send requests for WMS image and saves the image as a png file.
  • KMLConnection: Connects to the tweet database, retrieves the necessary information from it, generates the KML file, which embeddeds the image and the tweets in the same file, using groundOverlay, writes the KML string into a KML file and opens it in Google Earth.
  • GoogleEarthTweetMapper: Two lines of code calling the methods created in the previous classes. That is, we call the other two classes in the main class to implement it. This was done to ensure modularity of the code.
